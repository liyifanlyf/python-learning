2.1 基本语法
1）缩进：表示Python间的包含和层次关系
  用在if，while，for，def，class等关键字里面
2）注释：
  注释会被编译器或解释器略去
  文档注释是三个单引号或三个双引号（可以多行注释）
  注释能用#，也可以用引号
3）续行符：
  续行符用\表示。可以用于单行代码较长的情况，也适合对代码进行多行排版增加可读性的情况。（用了必须直接换行）
