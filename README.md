4.1 程序流程图
程序流程图是一种表达程序控制结构的方式。由一系列图形，流程线和文字说明等组成。流程图的七种元素如下：
1）起止符：表示程序逻辑的开始或结束
2）判断框：表示一个判断条件，并根据判断结果选择
3）处理框：表示一组处理过程，对应顺序执行的程序；逻辑
4）输入/输出框：表示程序的数据输入或结果输出
5）注释框：表示程序的注释
6）流向线：表示程序的控制流，以带箭头直线或曲线表达程序的执行路径
7）连接点：表示多个流程图的连接方式，常用于将多个较小流程图组织成较大流程图
描述一个计算问题的程序过程有多种方式，包括：IPO，流程图，伪代码或程序代码。其中，流程图采用图形方式最为直观，这里仅使用这种形式介绍3种控制结构的基本和扩展含义。
程序由3种基本结构组成：顺序结构，分支结构和循环结构
顺序结构是程序按照代码的编写顺序依次执行的一种运行方式

4.2 分支结构
1）单分支结构：
  分支结构是程序根据条件判断结果而选择不同向前执行路径的一种运行方式
语法格式：
  if<条件>：
    <代码块>
2）二分支结构：
  语句1是在条件为真时执行，语句2是在条件为假时执行。
语法格式：
  if<条件>：
    <代码块1>
  else:
    <代码块2>
3）多分支结构：
  多分支结构一般用于判断同一个条件或一类条件的多个执行路径。依次判断条件并执行第一个条件为True的语句。如果没有条件成立，执行else下面的语句。
if<条件>：
    <代码块1>
elif:
    <代码块2>
elif:
    <代码块3>
…………
else:
    <代码块N>
4）分支嵌套：
age=50
if age<18:
  print("未成年")
else:
  if 18<=age<30:
    print("年轻人")
  elif 30<=age<=60:
    print("中年人")
  else:
    print("老年人")

4.3 程序的循环
1.for循环：
    从遍历结构中逐一提取元素，放在循环变量中，对于每个提取的元素执行一次语句块。for语句的循环执行次数是根据迭代器中元素个数确定的。
语法格式：
  for<>in<>:
    <>
遍历结构可以是字符串，文件，range()函数或其它组合数据类型等。

range()函数的使用方式：
从a（包含）开始到b（不包含）结束，以c为差形成等差数列
for循环经常和range()函数一起使用，基本使用方式如下：
  for<循环变量>in range(<循环次数>):
      <语句块>
2.while循环
  程序执行while语句时，判断条件，如果为True，执行循环体语句，语句结束后返回再次判断条件；直到条件为False时，循环终止，执行与while同级别的后续语句。
语句格式：
    while<条件>:
      <代码块>
4.4 循环扩展及控制语句
1、循环扩展模块：
  当while循环正常执行之后，程序会绝续执行else语句中内容。else语句只在循环正常执行之后才执行并结束。
  语法格式：
            while<条件>：
                <语句块1>
            else:
                <语句块2>

2.循环控制语句：
1）continue语句：
  用来结束当前当次循环，即跳出循环体中下面尚未执行的语句，但不跳出当前循环。
2）break语句：
  跳出最内层循环，终止该循环后，从循环后的代码继续执行。

1.try...except:
    Python处理异常程序最简单的形式就是try...except...,类似于单分支选择结构。使用语法为：
  try：
      #可能会发生异常的代码，先执行以下试试看
  except<异常类型>：
      #如果try中代码抛出异常并被except捕获，执行此处代码
2.try...except...else...:
    该结构可以看为双分支结构，else中的代码只有在try中代码没有引发异常的情况下执行
使用语法为：
        try：
          #可能会发生异常的代码
        except<异常类型>：
          #如果try中代码抛出异常并被except捕获，执行此处代码
        else：
          #try代码正常执行，则执行此处代码
3.try...except...finally...
  该结构中，无论try代码中是否发生异常，finally语句块中的代码总会执行。
使用语法为：
        try：
          #可能会发生异常的代码
        except<异常类型>:
          #处理异常代码
        finally：
          #总是执行

5.4 集合（set）
1.定义：
  集合中的元素不可重复；
  元素类型只能是不可变类型；
  集合元素没有顺序，不能比较，不能排序；
  用花括号{}表示，没有索引和位置的概念；#不能用索引
  可使用set（）创建    #自动去掉重复元素
2.集合的操作符：
  S-T(差集)：返回新的集合，元素包含在S但不在T中
  S&T(交集)：返回新的集合，元素同时包含在S和T中
  S^T(补集)：返回新的集合，包含S和T中的非共同元素
  S|T(并集)：返回新的集合，包括S和T的所有元素
3.集合操作方法
  add（x）     增加新元素，如果该元素已经存在则忽略该操作，不会抛出异常
  remove（x）  删除集合中的元素，指定元素不存在则抛出异常
  clear（x）   移除集合中所有元素
  update（）   合并另外一个集合中的元素到当前集合中，并自动去除重复元素
  pop（）      随机删除并返回集合中的一个元素
  x in s      x是集合的元素，返回True，不是返回False
  x not in s  x不是集合的元素，返回True，是返回False
  discard（）  从集合中删除一个特定的元素，如果元素不在集合中则忽略该操作
4.集合运算
  Python集合还支持数学意义上的交集，并集，差集等运算。
a_set|b_set  #并集 <-->a_set.union(b_set)
a_set&b_set  #交集 <-->a_set.intersection(b_set)
a_set-b_set  #差集 <-->a_set.difference(b_set)
a_set^b_set  #对称差集<-->a_set.symmetric_difference(b_set)

7.2 函数的参数
1）位置传参：
    位置传参时时，实际参数（实参）与形式参数（形参）的对应关系是按位置来依次对应的调用函数时实参和形参的顺序需要严格一致，并且数量也必须相同。
    >>>def demo(a,b):#实参
               print(a,b)
    >>>demo(3,4)#形参#按照位置将实参3,4分别对应传递给形参a，b
    3,4
    >>>demo(3,4,5)#实参与形参的数量必须相同
    TypeError:demo() takes 2 positional arguments but 3 were given
#不能直接删空格退出函数体，要空格退出
#函数个数不对，直接报错
#函数参数如果不使用，不会报错

2）关键字传参：
    关键参数主要指调用函数时的参数传递方式，与函数定义无关
    指定参数名传参，实参顺序和形参顺序可以不一致
    >>>def data(name,age):
              print(name,age)
    >>>data("Jack","18")#按位置传参方式传递
    Jack 18
    >>>data(age="18",name="Jhon")#按关键字传参方式传递
    Jhon 18

3)传递参数时的序列解包：
    调用多个参数时，使用迭代对象作为实参，在实参名称前加“*”或“**”字符Python会自动解包，并按照相应的传参方式进行传参
        对列表、元组的解包
    >>>def demo(a,b,c):
    >>>lis=[2,3,4]
    >>>demo(*lis)#对列表进行解包，并按照位置传参
    24
    >>>tup=(3,4,5)
    >>>demo(tup)#对元组进行解包
    60
    #对形参进行解包

        对字典的解包
    >>>def fun(a,b,c):
              print(a,b,c)
    >>>p={'a':1,'b':2,'c':3)
    >>>fun(**p)#对字典进行解包，并按照关键字传参
    123
    #只能用**解包，且11对应
    #键需要是不可变的数据类型，字典里面的字母要变为字符串形式

4）单星号位置形参：
  单星号位置形参是指在函数创建过程中，用“*”将形参修饰，使得形参可以收集多于的位置参数形成元组
  说明：
        传参时，形参可以收集多于的位置参数形成元组
  >>>def fun(*p):
             print(p,type(p))
  >>>fun(1,2,3)
  (1,2,3)<class'tuple'>
  >>>fun([1,2,3])
  ([1,2,3],)<class'tuple'>
  >>>fun({1,2,3})
  ({1,2,3},)<class'tuple'>

5)双星号关键字形参
  双星号关键字形参是指在函数创建过程中，用“**”将形参修饰，使得形参可以收集多余的关键字函数
说明：
    ·传参时，形参可以收集多余的关键字参数形成字典
>>>def fun(**p):

7.3 函数的返回值及变量的作用
return：return语句用来结束函数并将程序返回到函数被调用的位置继续执行
        return语句可以出现在函数中的任何部分，并且可以存在任意个
        可以同时返回0个或多个函数运算的结果给函数被调用处的变量
        当return返回多个值时，返回的值形成元组数据类型

变量：局部变量：1、定义在函数内部的变量被称为局部变量（包括函数的形参）
               2、局部变量只能在函数内部使用
               3、局部变量在函数调用时才能够被创建，在函数调用之后会自动销毁
     全局变量：1、定义在函数外部，模块内部的变量称为全局变量
              2、全局变量所有函数都可以直接访问（但函数内部不能进行赋值或修改操作）
              3、全局变量在函数内部修改时，需要用global关键字声明，且该变量与外部变量全局变量同名
        列表这种可变函数可以进行修改
代码复用：
        模块化设计：通过函数的封装功能将程序划分为主程序，子程序和子程序间关系的表达。
        要求：紧耦合：尽可能合理划分功能模块、功能块内部耦合紧密；
              松耦合：模块间关系尽可能简单，功能块之间耦合度低
函数：程序的一种基本抽象方式，它将一系列代码组织起来通过命名供其他程序使用。

9.3 time库函数
    time库是Python提供的处理时间标准库。time库的功能主要分为3个方面：时间处理，时间格式化和计时。
time库的函数： time.time()
              time.localtime
              asctime(localtime([secs])
              time.sleep(secs)
              time.strftime(format[,t])
1、time.time()    #以数字形式返回当前时间
    意思：获取时间戳。
    解释：它返回的是从1970年1月1日到现在一共过了多少秒（通常带小数）。你可以把它想象成电脑内部记录时间的“秒表读数”。
    考点：常用来计算程序运行了多久。
2、time.localtime([secs])    #本地时间
    意思：将时间戳转换为本地时间的结构化数据。
    解释：它把一个秒数变成了一个包含年，月，日，时，分，秒等信息的“结构体”。
    考点：括号里的[secs]，不写就是当前时间
            #localtime(time())
3、asctime()    #固定格式
    意思：把结构体变成固定样子的字符串。
    考点：固定格式不能改！长的像 Tue Jul 21……。
          #asctime(localtime(1) #1时间戳  ‘Thu Jan 1 08:00:01 1970’
4、sleep()    #睡觉/暂停
    意思：让程序暂停几秒。
    大坑：它没有返回值（返回None）！
5、strftime()    #自定义格式
    意思：把结构体变成你想要的样子。
    大坑：最灵活，考试最爱考
